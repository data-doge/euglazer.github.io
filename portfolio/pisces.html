PISCES

Project Overview

>> The Planetary In-Situ Capillary Electrophoresis System (PISCES) is a portable, miniature, modular instrument designed to analyze molecular signatures of extraterrestrial life. The instrument uses microcapillary electrophoresis (µCE) with laser-induced fluorescence detection (LIFD) to analyze fluorescently-labeled analytes prepared on a multilayer microfluidic chip.
>> PISCES is divided into three main systems:
[1] The pneumatic system is the processing core of PISCES. It contains 48 electrofluidic solenoid valves which open and close together in various sequences to induce movement of fluids through an array of interconnecting channels on one layer of the microfluidic chip. On this layer of the chip, introduced samples are processed with a set of stored reagents before separation by µCE.
[2] The µCE system is the analytical core of PISCES. It contains an electrophoretic channel integrated into a separate layer of the microfluidic chip. By generating a strong electric field across the fluidic medium in the channel, separation of a chemical mixture into components of differing mass and charge is achieved.
[3] The LIFD system is the detection core of PISCES. It uses an optical pathway and spectrometer to detect laser induced fluorescence of labeled analytes passing through the electrophoretic channel.

Project Details

>> PISCES is divided into five modules, each of which can be easily detached from the system for subsystem testing in extreme environments mimicking the desired planetary target.
>> The pneumatic module contains the pneumatics and control circuitry required to operate the 48 electrofludiic solenoid valves responsible for on chip sample processing. This includes two micropumps (air and vacuum), pumping manifolds (to interface the micropumps with the solenoid valves), a printed circuit board (to interface the control circuitry with the solenoid valves), an Arduino microcontroller (to switch the solenoid valves between states), and an h-bridge solenoid driver board (to convert the Arduino's unipolar digital output to the bipolar voltage input that the solenoids require).

The µ module contains the circuitry required to apply between 3-15 kV to the microdevice's electrophoretic channel. This includes two 3kV HV power supplies and one 15kV power supply, and circuitry allowing control over and monitoring of the output voltage for each power supply.
